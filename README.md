Billing System Web Application

🚀 A modern and efficient Billing System designed to simplify product billing, apply smart offers, and generate accurate totals in real-time.


📌 Project Overview

This project is a frontend-based billing system that allows users to:

- Select products
- Adjust quantities dynamically
- View real-time bill summary
- Apply automatic discounts & offers

It demonstrates strong fundamentals of UI design, state management, and logical implementation of pricing rules.

✨ Key Features

✔️ Dynamic product quantity management
✔️ Real-time subtotal & final total calculation
✔️ Smart discount system (BOGO & percentage offers)
✔️ Clean and intuitive user interface
✔️ One-click reset functionality
✔️ Responsive layout for better user experience


🛠️ Tech Stack
-React
-Redux + Redux Toolkit
-TypeScript
-Hooks (useMemo, useSelector, useDispatch)
-Custom billing logic
-HTML/CSS

🧠 Business Logic Implemented

- 🧀 Cheese BOGO Offer → Buy 1 Get 1 Free
- 🍞 Bread Discount → 50% off on selected quantity
- 🧈 Butter Discount → 33% price reduction

💡 The system automatically detects eligible offers and applies them in the final bill.
